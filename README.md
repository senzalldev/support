## How to Get Help

### Report a Bug
1. Go to the [Issues](https://github.com/senzalldev/support/issues) tab
2. Click **New Issue**
3. Choose the **Bug Report** template
4. Include which app, what happened, and what you expected

### Request a Feature
1. Go to the [Issues](https://github.com/senzalldev/support/issues) tab
2. Click **New Issue**
3. Choose the **Feature Request** template
4. Describe what you'd like to see

### Ask a Question
1. Go to the [Discussions](https://github.com/senzalldev/support/discussions) tab (if enabled)
2. Or create an issue with the **Question** label
